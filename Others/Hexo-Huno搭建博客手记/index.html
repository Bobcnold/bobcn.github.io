Hexo-Huno搭建博客手记

原来hexo-renderer-markdown-it是可以做到markdown和html混编的。（2015-11-28） 简单介绍了hexo、主题、语法高亮和renderers。（2015-11-24）

Hexo官方网站

可前往 https://hexo.io 查看如何使用hexo。

$ npm install -g hexo-cli
$ npm install hexo --no-optional

$ hexo init <folder>
$ cd <folder>
$ npm install

Huno主题

此博客是在huno主题的基础上修改的。huno在github上的主页是 https://github.com/someus/huno 。

知乎上有一篇《有那些好看的hexo主题？》介绍了很多主题。当然，在hexo的网站上就提供了很多主题。

语法高亮

  1. 在<head>标签中添加

     <link href="http://cdn.bootcss.com/highlight.js/8.0/styles/monokai_sublime.min.css" rel="stylesheet">

    可前往 https://highlightjs.org 下载各种样式的css。

  2. 在</body>标签前添加

     <script src="http://cdn.bootcss.com/highlight.js/8.0/highlight.min.js"></script>
 <script >hljs.initHighlightingOnLoad();</script>

Other Renderers

  • hexo-renderer-marked

    默认的renderer，支持和html标签混编，但是对于markdown语法的解析没有hexo-renderer-markdown-it好，有些格式不能被正确解析。

  • hexo-renderer-markdown-it

    对于markdown语法的解析比较好，但是默认不支持和html标签混编，当然也就没办法通过<!--more-->来生成文章列表中的预览内容了。

    还好，找到有一篇文章介绍了可以在hexo的_config.yml文件中添加如下配置，以支持和html的混编。

      markdown:
    render:
      html: true

卸载原renderer，使用新的renderer

$ npm un hexo-renderer-markdown-it --save
$ npm i hexo-renderer-marked --save
