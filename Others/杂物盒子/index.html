Bob的杂物盒子

来不及、懒得整理的东西暂时放在这里。

  • ssh连接服务器显示

      @@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
  @    WARNING: REMOTE HOST IDENTIFICATION HAS CHANGED!     @
  @@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@

    找到保存的相应ip的密钥，删除即可。

      $ cd ~/.ssh
  $ vim known_hosts

  • 修改了文件，然后push到github却发现没有改变。原来是git默认不区分大小写。

    通过下命令，修改git对大小写敏感。

      git config core.ignorecase false

  • 文件权限

    drwxr-xr-x

    d 文件类型：d是目录文件，l是链接文件，-是普通文件，p是管道 rwx 文件所有者的权限：r是读，w是写，x是执行 r-x 与文件所有者属于同一个组的用户的权限： r-x 其他用户的权限

  • brew

    brew 又叫Homebrew，是Mac OSX上的软件包管理工具，能在Mac中方便的安装软件或者卸载软件， 只需要一个命令， 非常方便。brew的官方网站：http://brew.sh/。
