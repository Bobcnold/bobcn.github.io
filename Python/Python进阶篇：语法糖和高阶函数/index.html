学习Python——进阶篇：语法糖和高阶函数

参考廖雪峰的官方网站中的《Python教程》学习Python记录的笔记。这是第二篇。

一、语法糖

切片

可用于list、tuple和String。

>>> aList = ['Rabbit', 'Fox', 'Dog', 'Cat', 'Pig']
>>> aList = aList[1:3:2] # 起始（含）、（小于）长度、步长 aList = ['Fox']
>>> anotherList = aList[:] # 复制了一个list

列表生成器

[<元素生成规则表达式> <变量1取值（范围及条件）> <变量1取值（范围及条件）> ...]

>>> [m.lower() + str(n) for m in 'ABC' for n in range(0,4) if n % 2 == 0]
['a0', 'a2', 'b0', 'b2', 'c0', 'c2']

生成器generator

与列表生成器不同之处在于：

  • 形式上：[]换成()
  • 实现上：列表生成器会直接生成list并保存在内存中；而生成器保存的是算法，只有通过next(generator)调用时，才会计算并返回下一个值，如果已经是最后一个元素了，则会抛出StopIteration异常。
>>> g = (x * x for x in range(2, 10))
>>> next(g) # =>4
>>> next(g) # =>9
>>> for n in g: print(n)

yield将函数变为生成器。yield类似于我们debug程序时的一个断点。next(g)时会返回yield后面的表达式的值，下次运行继续运行“断点”之后的语句，直到遇到下一个“断点”。

>>> def square():
>>>     n = 1
>>>     while(True):
>>>         yield n * n
>>>         n = n + 1
>>> p = printInt()
>>> next(p) # => 1
>>> next(p) # => 4

生成器这个东西还蛮有趣的。

迭代（遍历）、迭代器

>>> isinstance('abc', Iterable) # 查看对象是否可迭代
>>> for x in aList: pass
>>> for key in aDict: pass
>>> for value in aDict.values(): pass
>>> for key, value in aDict.items(): pass
>>> for index, value in enumerate(aList): pass
>>> for index, value in [(0, 'Fox'), (1, 'Dog')]: pass

  • generator和带有yield的generator function也是iterable（可迭代）对象。

  • iterator迭代器是指可以通过调用next()返回下一个值的对象。

    python的for循环的本质就是通过next()来实现的。非iterator的iterable对象（如list）可以通过iter()转化为iterator对象。

二、高阶函数

method_name(f, iterable) # f为调用函数名

map、reduce

map会对iterable中的每一个元素执行函数f，并返回一个由执行结果组成的iterator：

>>> def f(x): return x * x
>>> r = map(f, [1, 2, 3, 4]) # map返回的值，即r是一个Iterator
>>> list(r) # => [1, 4, 9, 16]

reduce将iterable中的前两个元素作为函数f的参数，再将返回结果放在iterable的首位，反复进行，最终返回一个结果：

>>> def add(x, y): return x + y
>>> reduce(add, [1, 2, 3, 4]) # =>10

filter

filter会对iterable中的每一个元素，按照函数f返回的结果进行判断，如果返回True则保留在返回的iterator中。

>>> def is_even(x):
>>>     return x % 2 == 0
>>> r = filter(is_even, [1, 2, 3, 4, 5, 6,7])
>>> list[r] # => [2, 4, 6]

教程中的求素数的例子很好的展现了这个方法的应用。

sorted

>>> sorted([-7, -2, 4, 1, 3], key=abs, reverse=True)

排序，后两个为可选参数。
