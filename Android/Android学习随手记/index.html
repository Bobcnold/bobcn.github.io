  2. Layout

    layout_: 在布局中的属性

    1. LinearLayout

      orientation: horizontal vertical

      layout_gravity: horizontal只可以竖直方向对其，反之亦然

      layout_weight:按照权重划分“剩余”空间（除去已经指定好的尺寸）

    2. RelativeLayout

      layout_align(ToParent)...: 左对齐右对齐同时设置会使两个控件等宽

      layout_center...

      layout_(above below toLeftOf toRightOf toStartOf toEndOf): toStartOf和toLeftOf有啥区别？

    3. FrameLayout

      layout_gravity: right|top

    4. TableLayout

      stretchColumns可拉伸列 shrinkColumns可收缩列 collapseColumns可折叠列

      TableRow(强制宽match_parent高wrao_content 可不写)，内容都是wrao_content

      layout_column layout_span: 占据的列数

    5. AbsoluteLayout
