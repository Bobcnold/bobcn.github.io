Laravel安装及设置

Laravel安装、新建工程，设置环境变量；以及配置Apache使用路由。

  1. First, download the Laravel installer using Composer:

    composer global require "laravel/installer"

  2. Create project:

    cd <directory> composer create-project --prefer-dist laravel/laravel <project name>

  3. Environment Configuration:

    工程根目录下有环境变量配置文件。

    • .env中配置环境变量；
    • .env.example用于多人协作时指明必须配置的环境变量；

    APP_ENV的值配置成testing则会使用.env.testing中的配置。

  4. Access or set the configuration values using the global config helper function:

    所有配置文件都存放于config目录下。配置文件中第二个参数是.env中没有APP_DEBUG这个key时返回的值。

     'debug' => env('APP_DEBUG', false),

    可以通过如下方式获取或设置配置值。

     $value = config('app.timezone');
 config(['app.timezone' => 'America/Chicago']);

  5. 需要开启Apache 的mod_rewrite功能才能使用路由

     $ cd /etc/apache2
 $ sudo vim httpd.conf

     #LoadModule rewrite_module modules/mod_rewrite.so

 ...

 <Directory />
     Options FollowSymLinks
     AllowOverride None
 </Directory>

    把#LoadModule rewrite_module modules/mod_rewrite.so 前的#去掉

    将AllowOverride None 改成 AllowOverride ALL

    重启Apache

     $ sudo apachectl restart
