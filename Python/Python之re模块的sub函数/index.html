Python之re模块的sub函数

了解了一下python中使用正则匹配到相应字符串，并使用其他字符串替换的方法：re.sub()。

Python的sub函数：替换字符串

re.sub()共有5个参数：

  • 3个必选参数：pattern(正则字符串)、repl(提供用做替换的字符串)、string(被处理字符串)
  • 2个可选参数：count(很显然这是处理的个数)、flags
repl
>>> import re

repl可以是一个字符串，

>>> str = "hello world hello bob !"
>>> pattern = re.compile(r"(hello)\s?\w+\s?\1\s?bob")
>>> result = pattern.sub("\g<1>", str)
>>> result # => 'hello !'

也可以是一个函数：你可以对每一个匹配的对象做操作，这正式我求助于python的re的初衷！这简直太Cooooool啦！

>>> def addSuffix(matched):
>>>     suffix = matched.group("content")
>>>     suffix += ".superView"
>>>     return suffix
>>>     
>>> str = "CGRectMake(0, 0, kHeight(one), kHeight(another))"
>>> pattern = re.compile(r"(?<=kHeight\()(?P<content>\w+)(?=\))")
>>> result = pattern.sub(addSuffix, str)
>>> result
'CGRectMake(0, 0, kHeight(one.superView), kHeight(another.superView))'

IDE

在日常使用中，许多IDE也提供了类似的功能。如Xcode、Sublime Text等。

在Sublime Text中使用Shift+Command+F可以实现多文件的替换。

被替换字符串正则表达式：(?<=kHeight\()(\w+)(?=\))

替换字符串表达式：$1.superView

kHeight(contentView); kHeight(webView); => kHeight(contentView.superView); kHeight(webView.superView);
