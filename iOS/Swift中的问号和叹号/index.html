Swift中的?和!

Swift中的?和!有多种用法，看到一篇文章总结的不错。（2015-11-28）

?的几种用法：

Optional变量

表示这个变量可能是nil，而没有值。

var optionalVar: String?
Optional变量调用方法

如果该optional变量不为nil则调用该方法，否则跳过该语句。

optionalVar?.isEmpty
可选的（代理）方法
delegate.loginDidSuccess?(userId, token)
as?向下转型（Downcast）

注意：as?会返回optional类型值。下例中if let要求=右侧的值必须为optional值。

let imageView: UIView = UIImageView.init()
if let imageView = imageView as? UIImageView {
    imageView.image = UIImage.init(named: "i.png")
}

!的几种用法：

拆包（unwrap）

从optional变量中取出它的值

optionalVar!.isEmpty
声明Implicitly Unwrapped Optionals变量

一般用于类中的属性。表示该变量一定会有值，不需要显示初始化；最常见的就是已经在nib中添加好的控件（@IBOutlet ...）。

class MyView: UIView {
    var label: UILabel!
}
