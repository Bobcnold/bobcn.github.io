学习Python——基础篇

参考廖雪峰的官方网站中的《Python教程》学习Python记录的笔记。（2015-11-29）

〇、运行、输入输出

命令行运行python：

$ python
$ python3

运行python文件：

.py文件头

#!/usr/bin/env python3
# -*- coding: utf-8 -*-

授权并运行.py

$ chmod a+x hello.py
$ ./hello.py

输出：

>>> print('The quick brown fox', 'jumps over', 'the lazy dog')
The quick brown fox jumps over the lazy dog
>>> print(r'\\\t\\') # 不转义
>>> # 多行打印字符串
>>> print('''line1
... line2
... line3''')

输入：

>>> name = input(please enter your name:)

一、数据类型

数据类型：整型、浮点数、字符串、布尔值（True、False）、空值（None）

数据类型转换：

>>> int('1.23') # => 1.23

整型

>>> 10/3 # =>3.3333333333333335
>>> 10//3 # => 3

字符串：

>>> ord('中') # 显示字符的整数编码
>>> chr(200013) # 整数编码转换为字符

>>> 'abc'.encode('ascii') # string转换为bytes
>>> b'\xe4\xb8\xad\xe6\x96\x87'.decode('utf-8') # bytes转换为string

>>> len('中文') # 字符串长度（=>2）
>>> len('中文'.encode('utf-8')) # bytes长度（=>6）

>>> 'my name is %s, my age is %02d, my height is %.1fm.' % ('Bob', 18, 1.70) # 格式化字符串

二、数据容器

list、Tuple：

类似于Array。

>>> myFriends = ['Fox', 'Dog', 'Cat']
>>> myFriends[-3]
>>> myFriends.insert(1, 'Rabbit')
>>> myfriends.pop(3)
>>> myFriends.append('Pig')
>>> myFriends.pop()
>>> myFriends[0] = 'LittleFox'

Tuple的不同之处在于，一旦初始化就不能再修改。

>>> myFamilys = ('nana', 'mama', 'sis')
>>> onlyOne = (1, ) # 只有一个元素时，需要元素后面加逗号；否则onlyOne = 1

dict

类似于Swift中的Dictionary或java中的Map。

>>> myInfos = {'name': 'Bob', 'age': 18, 'height': 1.70}
>>> 'email' in myInfos # 判断dict中是否存在某个key（=>False）
>>> myInfos['name'] # 如果key不存在会报错
>>> myInfos.get('weight', 60.0) # 如果key不存在则返回None或者指定的value（=>60.0）
>>>

set

跟数学上的集合类似。

>>> aSet = set([1, 2, 3, 1]) # =>{1, 2, 3}
>>> aSet.add(2) # =>{1, 2, 3}
>>> aSet.remove(1) # =>{2, 3}
>>> aSet | set([1, 2]) # 并集（=>{1, 2, 3}）
>>> aSet & set([3, 4]) # 交集（=>{3}）

三、流程控制语句

if <条件判断>:
    <执行语句>
elif <条件判断>:
    <执行语句>
else:
    <执行语句>

for num in list(range(5)):
    sum += num

while  n < 5:
    sum += n

四、函数

>>> help(abs)
>>> a = abs; b = a(-1); b # =>1

定义函数

def my_abs(x):
    if not isinstance(x, (int, float)):
        raise TypeError('bad operand type')
    if x >= 0:
        return x
    else:
        return -x

空语句pass，类似于{}。

返回值

可以返回多个值（实际上返回的是一个tuple）：

>>> x, y = someFunc()

默认参数

默认参数必须指向不可变对象。（参见《函数的参数》）

>>> def enroll(name, gender, age=6, city='Beijing'): pass
>>> enroll('Bob', 'F', city='Shenzhen')

可变参数

>>> def my_sum(*numbers):
...     sum = 0
...     for num in numbers:
...             sum += num
...     return sum
>>> my_sum(1, 2, 3) # => 6

关键字参数

>>> def person(name, age, **others)
...     print('name:', name, 'age:', age, 'others:', others)
>>> person('Bob', 18, gender='F', city='Shenzhen') # =>name: Bob age: 18 others: {'gender': 'F', 'city': 'Shenzhen'}

命名关键字参数（应不常用）

“解list”、“解dict”

>>> params1 = ['Bob', 'F']
>>> params2 = {'age': 18, 'city': 'Shenzhen'}
>>> enroll(*params1, **params2)
>>> # “类似于”解指针，*可以“解list”，**可以“解dict”。上语句等价于：
>>> enroll('Bob', 'F', age=18, city='Shenzhen')
