Git学习笔记

学习廖雪峰《Git教程》的笔记，及实际应用的随记。

设置参数

设置userName和email

$ git config --global user.name "Bobcn"
$ git config --global user.email "xcode_hu@163.com"

全局忽略文件、文件夹

$ vim ~/.gitconfig
/* .gitconfig文件，excludesfile设置忽略文件列表文件 */
[user]
    name = bobcn
    email = xcode_hu@163.com
[core]
    excludesfile = /Users/bobcn/.gitignore

在目录下新建一个.gitignore文件，会对其所在目录及所在目录的子目录生效。

基础git操作

为当前目录创建git仓库

$ git init

添加修改到暂存区

$ git add <file name>
$ git add .

提交暂存区修改到仓库

$ git commit -m "description about change"

查看当前状态

$ git status

查看文件更改内容

$ git diff <file name> 比较工作区和暂存区
$ git diff HEAD -- <file name> 比较工作区和仓库

查看提交历史

$ git log

查看命令历史

$ git reflog

撤销操作

版本回退(HEAD表示最后一次commit的版本)

$ git reset --hard HEAD^
$ git reset --hard HEAD~100
$ git reset --hard <commit id>

丢弃暂存区的修改

$ git reset HEAD <file name>

丢弃工作区的修改

$ git checkout -- <file name>

删除文件(添加删除操作到暂存区)

$ git rm <file name>

分支操作

创建并切换分支

$ git checkout -b dev

创建分支

$ git branch dev

切换分支

$ git checkout dev

查看当前分支

$ git branch

合并分支内容

$ git branch master
$ git merge dev

删除分支

$ git branch -d dev

隐藏/恢复工作区

隐藏工作区（的修改）

$ git stash

查看被隐藏的工作区

$ git stash list

恢复工作区并从隐藏列表中删除

$ git stash pop

恢复工作区

$ git stash apply
$ git stash apply stash@{0}

从隐藏列表中删除

$ git stash drop

标签

新建标签

$ git tag v1.0.1
$ git tag v0.9.0 6224937
$ git tag -a v1.1.3 -m "add new function" 3628164

查看标签

$ git tag

查看标签说明

$ git show v1.0.1

删除标签

$ git tag -d v0.9.0

多人协作

尝试推送本地修改

$ git push origin dev

如果推送失败，表明远程分支比本地更新，需要合并到本地

$ git pull

如果合并冲突，则解决冲突，在本地提交后，再次推送

建立本地分支和远程分支的链接

$ git branch --set-upstream-to=origin/dev dev

推送标签至远程

$ git push origin v1.0.1

推送本地尚未推送的标签

$ git push origin --tags

删除远程标签

$ git push origin :refs/tags/v0.9.0
